SQL

SQL is the main language behind data manipulation. All of user data saved by the websites is stored in databases, and in order to communicate with those databases SQL is used by the webapps to either SELECT, INSERT, DELETE, or UPDATE information. While there are many other operations that can be done with SQL, webapps mostly just save or fetch data. Most other data manipulation happens on the backend during data analysis.
